# Java-Swing-Calculator
این پروژه یک ماشین‌حساب گرافیکی ساده است که با استفاده از Java Swing طراحی و پیاده‌سازی شده است. این برنامه امکان انجام عملیات پایه‌ای ریاضی مانند جمع، تفریق، ضرب و تقسیم را فراهم می‌کند و از اعداد اعشاری و منفی نیز پشتیبانی می‌کند.

برنامه شامل سه بخش اصلی است:

پنجره اصلی (JFrame)

نمایشگر عدد و متن (JTextField)

دکمه‌ها (JButton) و مدیریت رویدادها (ActionListener)

1. JFrame - پنجره اصلی

یک پنجره اصلی ساخته شده و عنوان آن "Calculator" است.

setDefaultCloseOperation مشخص می‌کند که وقتی کاربر روی دکمه X کلیک کرد، برنامه بسته شود.

setSize اندازه پنجره را تعیین می‌کند و setLocationRelativeTo(null) باعث می‌شود پنجره در وسط صفحه نمایش ظاهر شود.

2. نمایشگر عدد و متن (JTextField)

JTextField برای نمایش عدد فعلی یا نتیجه استفاده می‌شود.

setEditable(false) باعث می‌شود کاربر نتواند مستقیماً متن را تایپ کند و فقط با دکمه‌ها تغییر کند.

setHorizontalAlignment(SwingConstants.CENTER) متن را در مرکز نمایشگر قرار می‌دهد.

اندازه دلخواه با setPreferredSize تعیین شده است.

3. دکمه‌ها و پنل‌ها

   دکمه‌ها در دو بخش قرار دارند:

left شامل اعداد و دکمه "=" است.

right شامل عملیات ریاضی و دکمه "C" است.

از GridLayout برای مرتب کردن دکمه‌ها در جدول استفاده شده.

(5,1) یعنی ۵ ردیف و ۱ ستون برای عملیات
